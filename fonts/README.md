# Font Files

Place your font files in this directory with the following naming convention:

## Departure Mono
- `DepartureMono-Regular.woff2` (or `.woff`, `.ttf`, `.otf`)
- `DepartureMono-Bold.woff2` (or `.woff`, `.ttf`, `.otf`)

## Neue Machina
- `NeueMachina-Regular.woff2` (or `.woff`, `.ttf`, `.otf`)
- `NeueMachina-Bold.woff2` (or `.woff`, `.ttf`, `.otf`)

**Note:** `.woff2` format is preferred for best performance, but `.woff`, `.ttf`, or `.otf` formats will also work.

If your font files have different names or weights, you can update the paths in `app/fonts.ts`.
